Propriétés système Java à définir dans la commande ant qui lance le processus CLIF

Conseil: définir une seule propriété par ligne, avec la syntaxe classique propriete.nom=valeur

Attention: ces propriétés sont définies uniquement dans le processus ant, et ne sont pas propagées au processus CLIF lancé par ant. Pour définir des propriétés système dans le processus Java de CLIF, ajouter des expressions -Dproperty.name=value dans la section dédiées aux options Java.
