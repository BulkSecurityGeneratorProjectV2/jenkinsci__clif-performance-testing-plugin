Options de nettoyage statistique des mesures, consistant à exclure les valeurs les plus extrêmes.

Le facteur de conservation permet d'indiquer l'amplitude de conservation.
Soit :

  • m la moyenne des données
  • e l'écart type
  • f le facteur de conservation
Les données conservées sont celles comprises dans l'intervalle ]m - f*e, m + f*e[.

Le pourcentage de données à conserver permet de limiter le nettoyage afin de ne pas éliminer trop de mesures.
