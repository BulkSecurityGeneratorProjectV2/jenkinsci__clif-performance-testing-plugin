Permet de paramétrer la détection des actions en succès.

Par défaut, seul le champ succès des rapports de requête est utilisé pour déterminer si l'action est en succès ou en échec.
Pour permettre de détecter les erreurs de manière ad hoc, le plugin peut se baser sur l'analyse du champ r√©sultat ("result") des rapports de requête. Pour activier cette analyse, il suffit de définir une expression régulière caractérisant le succès d'une requête, pour un type de requête donné.
Le format des expressions régulières est celui des expressions régulières Java (voir java.util.regex.Pattern).
